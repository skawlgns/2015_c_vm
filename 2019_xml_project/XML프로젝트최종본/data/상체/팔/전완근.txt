- 전완근 : 전완은 아래팔을 뜻한다. 

- 운동이름 : 바벨 리스트 컬
#벤치 가장자리나 무릎 위에 전완부를 고정시키고 손목의 움직임만으로 동작을 실시한다.
1.무릎을 꿇고 벤치 앞에 앉는다.
2.바벨을 어깨너비보다 약간 좁은 상태로 손바닥이 위를 향하게 잡고 벤치 위에 전완부를 올려놓는다.
3.손목을 바닥 쪽으로 구부리면서 바벨을 내린다.
4.손목을 위로 올려 바벨을 들어올린다.
